## generate

```ts twoslash title="generate-signature.ts"
import type { Encoding, GenerateResult, OutputFormat, QRCodeOptions } from '@ikenxuan/qrcode';

declare function generate<E extends Encoding = 'binary'>(
  options: QRCodeOptions,
  format: OutputFormat,
  encoding?: E,
): GenerateResult<E>;
```

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `options` | `QRCodeOptions` | 二维码内容、尺寸、样式、背景、Logo 等配置。 |
| `format` | `OutputFormat` | `'svg'`、`'png'`、`'jpeg'`、`'webp'`。 |
| `encoding` | `'binary' \| 'base64'` | 输出编码，默认 `'binary'`。 |

## QRCodeOptions

```ts twoslash title="options.ts"
import type { QRCodeOptions } from '@ikenxuan/qrcode';

const options: QRCodeOptions = {
  data: 'https://example.com',
  size: 300,
  margin: 12,
  shape: 'square',
};

options;
// ^?
```

| 字段 | 说明 |
| --- | --- |
| `data` | 要编码的 URL、文本或业务字符串。 |
| `size` | 二维码尺寸，单位像素，默认 `300`。 |
| `margin` | 二维码外边距，单位像素。 |
| `shape` | 整体形状，方形或圆形。 |
| `dotsOptions` | 点阵样式、颜色或渐变。 |
| `cornersSquareOptions` | 定位角外层方块样式。 |
| `cornersDotOptions` | 定位角内部点样式。 |
| `backgroundOptions` | 背景色、透明背景、背景渐变和圆角。 |
| `image` | Logo 图片二进制数据。 |
| `imageOptions` | Logo 尺寸、留白和背景点阵处理。 |

## 颜色格式

颜色字段接受常见 CSS 颜色字符串，包括 Hex、RGB/RGBA、HSL/HSLA、CSS Color 4 空格语法、OKLab、OKLCH 和常见命名颜色。
