## generate

```ts twoslash title="generate-signature.ts"
import type { Encoding, GenerateResult, OutputFormat, QRCodeOptions } from '@ikenxuan/qrcode';

declare function generate<E extends Encoding = 'binary'>(
  options: QRCodeOptions,
  format: OutputFormat,
  encoding?: E,
): GenerateResult<E>;
```

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `options` | `QRCodeOptions` | 二维码内容、尺寸、样式、背景、Logo 等配置。 |
| `format` | `OutputFormat` | `'svg'`、`'png'`、`'jpeg'`、`'webp'`。 |
| `encoding` | `'binary' \| 'base64'` | 输出编码，默认 `'binary'`。 |

## QRCodeOptions

```ts twoslash title="options.ts"
import type { QRCodeOptions } from '@ikenxuan/qrcode';

const options: QRCodeOptions = {
  data: 'https://example.com',
  size: 300,
  margin: 12,
  shape: 'square',
};

options;
// ^?
```

| 字段 | 说明 |
| --- | --- |
| `data` | 要编码的 URL、文本或业务字符串。 |
| `size` | 二维码尺寸，单位像素，默认 `300`。 |
| `margin` | 二维码外边距，单位像素。 |
| `shape` | 整体形状，方形或圆形。 |
| `dotsOptions` | 点阵样式、颜色或渐变。 |
| `cornersSquareOptions` | 定位角外层方块样式。 |
| `cornersDotOptions` | 定位角内部点样式。 |
| `backgroundOptions` | 背景色、透明背景、背景渐变和圆角。 |
| `image` | Logo 图片二进制数据。 |
| `imageOptions` | Logo 尺寸、留白和背景点阵处理。 |

## ImageOptions

控制中心 Logo 的嵌入方式，需要与 `image`（Logo 二进制数据）配合使用。

```ts twoslash title="image-options.ts"
import type { QRCodeOptions } from '@ikenxuan/qrcode';

declare const logo: Uint8Array;

const options: QRCodeOptions = {
  data: 'https://example.com',
  image: logo,
  imageOptions: {
    imageSize: 0.22,
    margin: 8,
    hideBackgroundDots: true,
  },
};

options.imageOptions;
// ^?
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `imageSize` | `number` | Logo 占二维码的比例，范围 `0.0 ~ 1.0`。建议 `≤ 0.25`，过大可能导致无法扫描。 |
| `margin` | `number` | Logo 周围的留白像素。 |
| `hideBackgroundDots` | `boolean` | 是否挖空 Logo 区域下方的点阵，避免图案叠加干扰。 |

<Callout type="info">
  `image` 接收的是图片**二进制数据**（`Uint8Array`），而非 URL 或路径。Node 端可用
  `readFileSync('logo.png')` 读取，浏览器端可用 `new Uint8Array(await blob.arrayBuffer())`。
  由于二维码具备纠错能力，中心 Logo 不会破坏可扫描性，但仍建议生成后用 `scan` 自检。
</Callout>

## 颜色格式

颜色字段接受常见 CSS 颜色字符串，包括 Hex、RGB/RGBA、HSL/HSLA、CSS Color 4 空格语法、OKLab、OKLCH 和常见命名颜色。
