# 调用示例

## Rust 主题二维码

```ts twoslash title="rust-theme.ts" lineNumbers
import { generate } from '@ikenxuan/qrcode';

const image = await generate({
  data: 'https://github.com/ikenxuan/qrcode',
  size: 360,
  margin: 16,
  dotsOptions: {
    dotType: 'rounded',
    gradient: {
      colorFrom: '#b7410e',
      colorTo: '#dea584',
    },
  },
  cornersSquareOptions: {
    cornerType: 'extra-rounded',
    color: '#5f2614',
  },
  cornersDotOptions: {
    cornerType: 'dot',
    color: '#5f2614',
  },
  backgroundOptions: {
    color: '#fff7ed',
    round: 0.08,
  },
}, 'png');
```

## 品牌 Logo 二维码

在二维码中心嵌入品牌 Logo。得益于二维码的纠错能力，即便 Logo 覆盖了中心区域，
内容依旧可以被正常扫描解码 —— 生成后用 `scan` 自检一次最稳妥。

```ts twoslash title="brand-logo.ts"
// @types: node
import { readFileSync } from 'node:fs';
import { generate, scan } from '@ikenxuan/qrcode';

const logo = readFileSync('brand.png');
const data = 'https://github.com/ikenxuan/qrcode';

const image = await generate({
  data,
  size: 420,
  image: logo,
  imageOptions: {
    imageSize: 0.22,        // Logo 占二维码的比例，建议 ≤ 0.25
    margin: 8,              // Logo 四周留白
    round: 0.16,            // Logo 圆角比例，0.5 可裁剪为圆形
    hideBackgroundDots: true, // 挖空 Logo 下方点阵，避免叠加干扰
  },
}, 'png');

// 自检：确认嵌入 Logo 后依旧可解码
if (await scan(image) !== data) {
  throw new Error('嵌入 Logo 后二维码无法解码，请调小 imageSize');
}
```

## 在 React 中预览

```tsx twoslash title="QRCodePreview.tsx"
import React, { useEffect, useState } from 'react';
import { generate } from '@ikenxuan/qrcode/browser';

export function QRCodePreview() {
  const [src, setSrc] = useState('');

  useEffect(() => {
    let url = '';

    async function render() {
      const bytes = await generate({ data: location.href }, 'png');
      url = URL.createObjectURL(new Blob([bytes], { type: 'image/png' }));
      setSrc(url);
    }

    void render();
    return () => URL.revokeObjectURL(url);
  }, []);

  return <img alt="当前页面二维码" src={src} />;
}
```

## 批量生成不同点阵

```ts twoslash title="dot-types.ts"
import { DotType, generate } from '@ikenxuan/qrcode';

for (const dotType of Object.values(DotType)) {
  const image = await generate({
    data: `dot type: ${dotType}`,
    dotsOptions: { dotType },
  }, 'webp');

  console.log(dotType, image.length);
}
```

## 生成后立即验证

```ts twoslash title="verify.ts"
import { generate, scan } from '@ikenxuan/qrcode';

const expected = 'https://example.com/pay?id=2026';
const image = await generate({ data: expected }, 'png');
const actual = await scan(image);

if (actual !== expected) {
  throw new Error('QR code verification failed');
}
```
